# 智能组合秤订单填表功能 - 测试说明

## 一、功能概述

- **API 端点**: `POST /api/v1/closing-form/submit`
- **目标表**: `data_doc_collection_1`（text、metadata_、embedding 列）

## 二、启动服务

```bash
cd /path/to/project-yamato-shanghai
python main.py
# 或
uvicorn main:app --host 0.0.0.0 --port 8000
```

### 验证数据库写入

```sql
SELECT id, text, metadata_, embedding IS NOT NULL AS has_embedding
FROM data_doc_collection_1
ORDER BY id DESC
LIMIT 5;
```

## 四、依赖说明

- BGE-M3 嵌入服务需可访问（`BGE_M3_API_URL`）
- PostgreSQL 需已创建 `data_doc_collection_1` 表（由 LlamaIndex PGVectorStore 首次写入时自动创建）

# 用户登录接口 - 前端接入指南

## 概览

后端提供 JWT（JSON Web Token）认证机制。前端通过 **用户名 + 密码** 登录获取 Token，
后续请求在 `Authorization` Header 中携带 Token 即可访问受保护的接口。

---

## 1. 登录

### `POST /api/v1/auth/login`

**Request Body**（JSON）：

```json
{
  "username": "admin",
  "password": "admin123"
}
```

**Response**（200 OK）：

```json
{
  "access_token": "eyJhbGciOiJIUzI1NiIs...",
  "token_type": "bearer"
}
```

**错误响应**：

| 状态码 | 场景 |
|--------|------|
| 401 | 用户名或密码错误 |
| 403 | 账户已被禁用 |
| 422 | 请求体格式错误 |

### curl 示例

```bash
curl -X POST http://localhost:8000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "admin", "password": "admin123"}'
```

---

## 2. 获取当前用户信息

### `GET /api/v1/auth/me`

需要在请求头中携带登录时获取的 Token：

```
Authorization: Bearer <access_token>
```

**Response**（200 OK）：

```json
{
  "id": 1,
  "username": "admin",
  "name": "管理员",
  "email": "admin@example.com",
  "phone": null,
  "department": null,
  "avatar": null,
  "is_active": true,
  "roles": []
}
```

**错误响应**：

| 状态码 | 场景 |
|--------|------|
| 401 | Token 缺失、过期或无效 |

### curl 示例

```bash
curl http://localhost:8000/api/v1/auth/me \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIs..."
```

## 3. 前端接入建议

### Token 存储

推荐使用 `localStorage` 或 `sessionStorage` 保存 `access_token`：

```javascript
// 登录后保存
localStorage.setItem('access_token', response.data.access_token);

// 读取
const token = localStorage.getItem('access_token');
```

### Axios 请求拦截器（推荐）

```javascript
import axios from 'axios';

const api = axios.create({ baseURL: '/api/v1' });

api.interceptors.request.use((config) => {
  const token = localStorage.getItem('access_token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// 响应拦截 - Token 过期时跳转登录页
api.interceptors.response.use(
  (res) => res,
  (err) => {
    if (err.response?.status === 401) {
      localStorage.removeItem('access_token');
      window.location.href = '/login';
    }
    return Promise.reject(err);
  }
);
```

### fetch 示例

```javascript
const res = await fetch('/api/v1/auth/me', {
  headers: {
    'Authorization': `Bearer ${localStorage.getItem('access_token')}`,
  },
});
const user = await res.json();
```

---

## 4. Token 说明

| 参数 | 值 |
|------|-----|
| 算法 | HS256 |
| 有效期 | 60 分钟（可通过环境变量 `ACCESS_TOKEN_EXPIRE_MINUTES` 调整） |
| Payload `sub` | 用户 ID（字符串） |

Token 过期后需重新调用登录接口获取新 Token。
